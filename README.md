# MemoryMap

MemoryMap — веб-приложение для сохранения воспоминаний о посещённых местах.

Пользователь может:
- добавлять новые места;
- указывать координаты;
- прикреплять описание;
- добавлять фотографии;
- просматривать свои воспоминания.

## Запуск проекта

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Применение миграций

```bash
python manage.py makemigrations
python manage.py migrate
```

### Запуск сервера

```bash
python manage.py runserver
```

После запуска приложение доступно по адресу:

```text
http://127.0.0.1:8000/
```
